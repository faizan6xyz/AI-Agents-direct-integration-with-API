Set up credentials :

    1. Go to Google Cloud Console → create/select a project
    2. Enable Gmail API (APIs & Services → Library)
    3. Configure OAuth consent screen (External, add your Gmail as a test user)
    4. Create OAuth client ID credentials (Desktop app type) → download credentials.json

dependencies : 

    pip install google-auth-oauthlib google-auth-httplib2 google-api-python-client

Scopes :

    Non-sensitive (minimal review needed)
        https://www.googleapis.com/auth/gmail.addons.current.action.compose — manage drafts and send emails when interacting with the add-on
        https://www.googleapis.com/auth/gmail.addons.current.message.action — view email messages when interacting with the add-on
        https://www.googleapis.com/auth/gmail.labels — see and edit email labels

    Sensitive (require additional OAuth app verification)
        https://www.googleapis.com/auth/gmail.addons.current.message.metadata — view message metadata when the add-on is running
        https://www.googleapis.com/auth/gmail.addons.current.message.readonly — view email messages when the add-on is running
        https://www.googleapis.com/auth/gmail.send — send email on the user's behalf (no read/modify access)

    Restricted (require restricted-scope verification, plus a security assessment if you store/transmit the data)
        https://mail.google.com/ — full read, compose, send, and permanent delete access (only needed if you must bypass trash for permanent deletion)
        https://www.googleapis.com/auth/gmail.readonly — view messages and settings
        https://www.googleapis.com/auth/gmail.compose — manage drafts and send emails
        https://www.googleapis.com/auth/gmail.insert — insert emails into the mailbox
        https://www.googleapis.com/auth/gmail.modify — read, compose, send (no permanent delete)
        https://www.googleapis.com/auth/gmail.metadata — view metadata (labels, headers) but not the email body
        https://www.googleapis.com/auth/gmail.settings.basic — see/edit email settings and filters
        https://www.googleapis.com/auth/gmail.settings.sharing — manage sensitive mail settings (admin-only, domain-wide delegation)
