Step-by-step

    1. Meta Developer account + app
        Go to developers.facebook.com → create a Business-type app.
        Add the "Instagram Graph API" product to the app.

    2. Configure OAuth
        Set up OAuth 2.0 with Facebook Login for Business (most devs use this over plain Business Login since it integrates with Business Manager, useful if you'll manage multiple accounts later).
        Request the permission scopes you actually need:
        instagram_basic — read profile/media
        instagram_manage_comments — reply/hide/delete comments
        instagram_manage_messages — DMs
        instagram_manage_insights — analytics

    3. Auth flow
        Implement the authorization code flow → exchange code for a short-lived token → exchange that for a long-lived Instagram User access token (60 days, refreshable).

    4. Test mode (before review)
        You get up to 25 test users without needing App Review — good enough to build and sanity-check your whole pipeline before submitting.

    5. Submit for App Review
        Required for production access beyond those 25 test users. Full process — setup through approval — typically takes 4-6 weeks, so start this early, in parallel with building.

    6. Basic call structure once you have a token:
        GET https://graph.facebook.com/v21.0/{ig-user-id}?fields=username,media_count,followers_count&access_token={token}
    
    7. Rate limits to design around
        200 calls/hour per connected Instagram account (not global — this trips people up). Batch requests where you can (e.g., fetching insights for 10 media items via one batch call instead of 10 separate calls).
    
    8. Webhooks (skip polling)
        For comments/DMs, set up a webhook endpoint (public HTTPS URL, POST receiver) instead of polling — this is the standard pattern and keeps you well under rate limits.

Scopes :

    If you use Business Login for Instagram (user logs in with Instagram credentials directly, no Facebook Page needed):
        instagram_business_basic — read profile and media
        instagram_business_content_publish — publish photos, videos, reels, carousels
        instagram_business_manage_comments — moderate comments
        instagram_business_manage_messages — send/receive DMs

    If you use Facebook Login for Business (Instagram account must be linked to a Facebook Page):
        instagram_basic — read profile and media
        instagram_content_publish — publish content
        instagram_manage_comments — comment moderation
        instagram_manage_insights — impressions, reach, audience demographics
        instagram_manage_messages — DMs (via the Messenger Platform)
        pages_show_list — list the Pages the user manages
        pages_read_engagement — read Page engagement data
        plus the Instagram Public Content Access feature if you need hashtag search
